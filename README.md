# ⬡ flow-mind

> 感情の物理モデルで動く、因果グラフ × 本音抽出 PWA

**flow-mind** は「自分が今何を感じているか」を可視化するためのツールです。ノードをタップするたびに神経発火のシミュレーションが走り、グラフ全体の同期パターンからリアルタイムで感情状態（attractor）を検出します。

---

## 感情の物理モデル（4階層）

flow-mindの中核となる設計思想です。感情は4つの物理層が積み重なった「同じ現象の別の顔」として実装されています。

| 層 | 物理現象 | flow-mind上の実装 |
|---|---|---|
| ① | イオンの流れ（電荷） | ノードタップ → `charge` 増加、時間減衰 |
| ② | 電流 → 電場・磁場 | エッジ上の電流パルス、Hebbian conductance 強化 |
| ③ | 神経同期（振動） | 時間窓内の同時発火クラスタ → 周波数帯推定（γ/β/α/θ/δ） |
| ④ | クオリア（主観的感情） | attractor収束 → 感情ラベル動的判定 |

---

## 機能

- **因果グラフ編集** — ノードの追加・接続・重み設定、ドラッグ移動、ピンチズーム
- **感情の物理シミュレーション** — タップで神経発火、電荷伝播、周波数帯別の波紋エフェクト
- **attractor検出** — 現在の同期パターンから「集中」「不安」「リラックス」などを動的判定
- **PageRankスコア** — エッジの重みと構造から各ノードの影響力を計算、上位5件をバー表示
- **AI分析（4モード）** — 構造分析 / 弱点検出 / ノード提案 / **本音抽出**（attractor情報をコンテキストとして渡す）
- **エコシステム連携** — `is_transfer` LocalStorage経由で他アプリとデータ交換、`#import=` URLハッシュでグラフ受信
- **PWA対応** — オフライン動作、ホーム画面追加

## 使い方

### ノード操作

| 操作 | 動作 |
|---|---|
| 空白タップ（追加モード） | 新規ノード作成 |
| ノードタップ | 発火 + 編集シート表示 |
| ノードドラッグ | 移動 |
| エッジタップ（編集モード） | エッジ削除 |
| ピンチ | ズーム |

### 感情状態の読み方

ノードをタップ → 連続してタップ → 画面上部のattractorバーに感情状態が表示される。複数のノードを素早く繰り返しタップするほど同期が深まり、より明確な帯域に収束する。

### AI 本音抽出

`✦ AI分析` → `🧠 本音抽出` — 現在検出中のattractor（同期帯域・密度・持続時間・関与ノード）をコンテキストとしてAIに渡し、まだ言語化できていない本音を仮説として抽出する。

## 設定

APIキーは他アプリ（quick-ref等）の設定画面から `localStorage` に保存されているものをそのまま参照します。

| キー | 用途 |
|---|---|
| `ml_claude` | Claude API key |
| `ml_gemini` | Gemini API key |
| `ml_openai` | OpenAI API key |
| `kansei_model` | 使用モデル（`claude` / `gemini` / `openai`） |
| `kansei_proxy_url` | Cloudflare Workers プロキシURL |
